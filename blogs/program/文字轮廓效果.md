## 背景

晚上在家游戏，朋友发了张图问我改如何用`css`实现


## 初步实现

一开始想到是便是使用`text-shadow`来做，很快就有了以下的代码

::: demo
<template>
  <div class="container">
    <p>hello world!!!</p>
  </div>
</template>

<style scoped>
.container {
  background: #000000;
  color: #ff0000;
  text-align: center;
}

p {
  text-shadow: 
    2px 0 0 #ffffff,
    -2px 0 0 #ffffff,
    0 2px 0 #ffffff,
    0 2px 0 #ffffff;
  font-size: 2em;
}
</style>
:::
